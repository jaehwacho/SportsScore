# PRD — 2026 춘계중고배구연맹전 단양대회 여고부 결과 사이트

| 항목 | 내용 |
|---|---|
| 문서 버전 | v1.0 |
| 상태 | 검토용 초안 |
| 대상 대회 | 2026 춘계중고배구연맹전 단양대회 (여고부) |
| 데이터 출처 | 한국중고배구연맹 / ㈜호각 |
| 연결 데이터 | `danyang_2026_data.json` |

---

## 1. 배경 & 목적

2026 춘계중고배구연맹전 단양대회(여고부)의 경기 데이터를 누구나 쉽게 확인할 수 있는 웹페이지로 제공한다. 현재 데이터는 PDF 형태로만 존재해 접근성과 가독성이 떨어지고, 특히 모바일에서 표를 보기 어렵다. 이를 PC·모바일 모두에서 잘 보이는 결과 사이트로 전환하는 것이 목적이다.

대회는 2026년 3월 13일(금)부터 3월 19일(목)까지 진행되었으며, 9개 팀이 총 14경기를 치렀다. 우승은 선명여고, 준우승은 중앙여고다.

## 2. 목표 & 성공 지표

- 대회 일정·결과·순위·선수 기록을 한 곳에서 확인 가능하게 한다.
- 모바일에서 가로 스크롤 없이 핵심 정보를 열람할 수 있게 한다.
- 원하는 경기·팀·선수를 3번의 탭 이내로 찾을 수 있게 한다.
- (정량 예시) 모바일 첫 화면 로딩 2초 이내, 주요 페이지 간 막힘 없는 이동.

## 3. 타겟 사용자

| 그룹 | 주요 니즈 |
|---|---|
| 선수·학부모 | 우리 팀·선수의 경기 결과와 개인 기록 확인 |
| 일반 배구 팬 | 대진표, 우승팀, 주요 경기 흐름 파악 |
| 연맹·관계자 | 공식 기록 열람 및 외부 공유용으로 정리된 화면 |

세 그룹을 모두 만족시키기 위해, "한눈에 보는 요약"과 "깊게 파는 상세 기록"을 화면 층위로 분리한다.

## 4. 범위 (Scope)

**포함 (In scope)**

1. 경기 일정·결과 (14경기, 세트 스코어 포함)
2. 대진표 (6강 → 4강 → 결승 토너먼트 브래킷)
3. 팀 순위·종합 기록 (9개 팀)
4. 선수 개인 기록·검색 (출전 선수 103명)

**제외 (Out of scope, v1)**

- 실시간 라이브 스코어, 영상·하이라이트, 로그인·댓글, 다국어 지원

## 5. 정보 구조 (IA) / 화면 구성

| 화면 | 핵심 내용 | 드릴다운 |
|---|---|---|
| 홈(요약) | 대회 개요, 우승·준우승, 최종 대진표 요약, 빠른 이동 메뉴 | — |
| 일정·결과 | 날짜별 14경기 리스트, 세트 스코어 | 경기 상세(세트별 점수) |
| 대진표 | 6강 → 4강 → 결승 브래킷 시각화 | 해당 경기로 이동 |
| 팀 순위 | 종합 순위표 (승패·세트·득실·최종성적) | 팀 상세(경기 이력) |
| 선수 기록 | 검색·필터·정렬 가능한 선수 테이블 | 선수 상세(신체정보+스탯) |

## 6. 기능 요구사항

### F1. 경기 일정·결과
- 14경기를 날짜(3/13~3/19)·경기번호 순으로 표시한다.
- 각 경기에 양 팀명, 세트 스코어(예: 3:1), 세트별 점수, 라운드(예선/6강/4강/결승)를 표기한다.
- 경기 상세에서 세트별 점수표를 제공한다.

### F2. 대진표
- 6강 → 4강 → 결승 구조의 브래킷을 표시하고, 우승팀(선명여고)을 강조한다.
- 모바일에서는 세로형·스크롤형으로 자동 전환한다.

### F3. 팀 순위·종합 기록
- 컬럼: 순위, 팀명, 경기수, 승-패, 세트 승-패, 득점, 실점, 최종 성적.
- 정렬을 지원하고 상위권(우승·준우승·4강)을 시각적으로 강조한다.

### F4. 선수 개인 기록·검색
- 전 팀 선수를 통합한 테이블(출전 103명)을 제공한다.
- 검색: 선수명 / 필터: 팀·포지션·학년 / 정렬: 득점·공격성공·블로킹·서브에이스·디그 등.
- 선수 상세: 신체정보(키·체중·포지션·학년·등번호)와 누적 스탯.

## 7. 데이터 모델

데이터는 사이트 코드와 분리된 `danyang_2026_data.json` 파일로 관리한다(향후 템플릿화 대비). 주요 구조는 아래와 같다.

| 영역 | 키 | 설명 |
|---|---|---|
| 대회 | `tournament` | 대회명, 일정, 출처, 데이터 고지문, 우승·준우승 |
| 순위 | `standings` | 9개 팀의 승패·세트·득실·경기당·세트당 득실·최종성적 |
| 경기 | `matches` | 14경기의 날짜·라운드·양 팀·세트 스코어·세트별 점수·승자 |
| 선수 | `players` | 111명(출전 103명)의 신체정보 + 누적 스탯 |

**선수 레코드 주요 필드**: `team, jersey, name, grade, pos1, pos2, height, weight, matches, sets, kills, total_attack, hit_pct, blocks, serve_aces, total_points, player_score, rcv_att, rcv_success, rcv_rate, digs, digs_per_set, played`

값이 없는 칸은 카운트 스탯의 경우 `0`, 비율(Hit%·리시브율)의 경우 `null`로 정리했다.

## 8. 비기능 요구사항

- **반응형**: 모바일(≤480px)·태블릿·PC 모두 대응. 넓은 표는 모바일에서 카드형 또는 가로 스크롤로 처리한다.
- **성능**: 정적 사이트 기준 첫 로딩 2초 이내.
- **접근성**: 충분한 폰트 크기·명도 대비, 표 헤더 고정.
- **브라우저**: 최신 Chrome·Safari·삼성인터넷 기준.

## 9. 디자인 방향 (제안)

- 깔끔한 스포츠 데이터 톤(흰 배경 + 포인트 컬러 1~2개), 한글 가독성 우선.
- 우승·순위 등 핵심 정보는 색과 아이콘으로 강조한다.

## 10. 제약사항 & 리스크

- **저작권 (최우선 선결 과제)**: 원본 PDF에 "연맹 데이터는 연맹의 자산이며 서면 허락 없이 복제·유통·가공·배포를 금지한다"는 고지가 있다. **공개 게시 전 한국중고배구연맹의 사용·게시 허가를 반드시 확인**해야 한다.
- **데이터 정확성**: PDF에서 수기로 옮긴 값은 게시 전 검수가 필요하다.

## 11. 향후 확장 (재사용 여부 미정 반영)

- v1은 단양대회 데이터로 완성하되, 데이터를 JSON으로 분리해 다음 대회 적용 시 데이터 파일만 교체하면 되도록 설계한다.
- 추후 옵션: 남고부·중등부 확대, 대회 간 비교, 라이브 스코어.

## 12. 미결정 사항 (Open Questions)

- 연맹 게시 허가 여부 및 데이터 고지문 노출 범위.
- 재사용 템플릿 여부(데이터 스키마 확정에 영향).
- 디자인 톤·브랜드 컬러(연맹 공식 가이드 유무).
